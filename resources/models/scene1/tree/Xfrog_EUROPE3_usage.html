XfrogPlants EUROPE 3

 

 

 

 

 

Due to the large size of the EUROPE 3 library, we have compressed the data using WinRAR.
Any program for Mac OS or Windows able to open a .rar file will work - please
uncompress first DVD 1 then DVD 2 and let them extract into the same folder.

 

 

 

Contents :

 

 

180  3D Models in one of: Maya, 3ds Max, Vue, Lightwave, Cinema 4D, .OBJ, .3DS formats

 

180  Models in XFR format, fully editable with Xfrog 3.5 Software
